# Малко повече за проекта:

<img width="50" height="30" alt="image" src="https://github.com/user-attachments/assets/3b2ad073-5c25-429d-8cc9-58fc395a8179" />

- Поддържа **2–4 играча**  
- Класически **UNO карти**:
  - Числови: 0–9          
 - Специални: `+2`, `Skip`, `Reverse`, `Wild`, `Wild+4`

<img width="300" height="300" alt="image" src="https://github.com/user-attachments/assets/054ca247-56c9-404d-ae3e-b1b074202bac" />

- **Интерактивно меню**:
  1. New Game – започва нова игра  
  2. Continue Saved Game – зарежда последната запазена игра  
  3. Exit – изход от програмата  
- Играчите могат да **запазват прогреса** и да **продължават по-късно**  
- Проверка за **валиден ход**  
- Забавни съобщения 

# Функционалности 
<img width="80" height="37" alt="image" src="https://github.com/user-attachments/assets/2b41a35a-687a-444a-ad6f-d7435dd9ca9f" />

- **Разбъркване на тестето** и автоматично теглене, ако няма валиден ход  
- **Специални карти**:
  - `+2`       → Следващият играч тегли 2 карти  
  - `Reverse`  → Променя посоката на играта  
  - `Skip`     → Следващият играч пропуска ход  
  - `Wild`     → Играчът избира цвят  
  - `Wild+4`   → Играчът избира цвят + следващият играч тегли 4 карти  
- **Правило UNO**: При последна карта, извикай "UNO", иначе получаваш наказателна карта  
- **Запазване и зареждане на игра** чрез `uno_save.txt`
